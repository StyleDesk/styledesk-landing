# Instagram Content — Feature Videos

Production playbook for StyleDesk.ai feature videos on Instagram (Reels + feed posts).

The goal: make feature walkthroughs look like a marketing team produced them, using only **Tella** (recording) and **Descript** (editing).

## Files in this folder

- `tella-recording-guide.md` — how to record screen + camera in Tella
- `descript-editing-guide.md` — how to polish audio, captions, and cuts in Descript
- `feature-video-template.md` — shot-by-shot template for a single feature video
- `instagram-specs.md` — formats, durations, and aspect ratios for Reels / posts / Stories

## Workflow at a glance

1. Write the script (4 lines is enough — never go unscripted)
2. Record in **Tella** with auto-zoom + camera bubble + branded background
3. Edit in **Descript**: Studio Sound → remove fillers → shorten gaps → captions → music
4. Export 1080×1920 MP4 for Reels (or 1080×1350 for feed)
5. Post to Instagram with a hook in the first frame and 3–5 hashtags
